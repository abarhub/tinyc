# tinyc

Interpreter for subset of c language.

- [x] lexer
- [ ] parser (work in progress)
- [ ] syntax analyser
- [ ] interpretor (work in progress)

syntax :
- [ ] type : int, string
- [ ] operator : +, -
- [ ] function : printf
- [ ] control structure : if, while


Example :
```C
void main(){
   int x=6;
   int y;
   y=8;
}
```
